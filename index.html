⚙ Setup Supabase

Ikuti langkah di bawah agar data portofolio tersimpan di cloud — bisa diakses siapapun, realtime.

1
Buka supabase.com → klik Start your project → daftar gratis pakai email atau GitHub.
2
Klik New Project → isi nama project & password database → klik Create new project. Tunggu ~1 menit.
3
Di sidebar kiri klik SQL Editor → klik New query → paste SQL di bawah ini → klik Run.
create table if not exists mv_works ( id uuid default gen_random_uuid() primary key, "ytId" text, title text not null, artist text, tags text[], thumb text, featured boolean default false, sort_order int default 0, created_at timestamptz default now() ); create table if not exists mv_site ( id int primary key default 1, data jsonb not null default '{}'::jsonb ); alter table mv_works enable row level security; alter table mv_site enable row level security; create policy "public read works" on mv_works for select using (true); create policy "public write works" on mv_works for all using (true); create policy "public read site" on mv_site for select using (true); create policy "public write site" on mv_site for all using (true);
4
Masuk ke Project Settings (ikon gear di sidebar) → klik tab API → copy Project URL dan anon public key.
5
Paste di bawah ini lalu klik Simpan & Mulai.

💡 URL dan Key disimpan di browser kamu. Data karya disimpan di Supabase cloud — semua pengunjung melihat data yang sama.

🔒 Password admin hanya tersimpan di browser kamu. Orang lain bisa melihat portofolio tapi tidak bisa menambah/menghapus karya.
